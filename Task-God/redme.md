Задание выходного дня
Напиши простое flask приложение которое умеет отдавать ping
Это приложение должно запускаться через uwsgi

Деплой сервиса: надо написать systems unit который запускает этот uwsgi/gunicorn
Uwsgi должен слушать на юникс сокете 
Также количество воркеров должно зависеть от Цпу на серваке 
Над uwsgi должен быть  nginx который проксирует запросы пользователей в сокет 


Все должно быть развернуто на двух тачках и должно быть отказоустойчиво
Для обеспечения отказоустойчивости между серверами должен быть настроен keepalived у которого есть динамическая проверка состояния сервиса 
При отключении uwsgi мастер ip должен переехать на тачку со здоровым uwsgi

Деплой всего соответсвенно через ansible
